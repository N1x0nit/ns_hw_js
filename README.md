# YOUR ENERGY

A single-page fitness application built with vanilla JavaScript and Vite.

## About

**YOUR ENERGY** is a SPA that helps users discover and explore exercises. Browse exercises by muscle groups, body parts, or equipment. Save favorites, rate exercises, and get inspired by a daily quote.

- 🏋️ Browse exercises filtered by Muscles, Body parts, or Equipment
- ⭐ Rate exercises and save favorites to localStorage
- 💬 Daily motivational quote via the API
- 📬 Newsletter subscription
